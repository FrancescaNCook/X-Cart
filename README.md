# X-Cart
MultiSafepay plugin for X-Cart 5.3

This plugin allows you to integrate all payment methods and giftcards offered by MultiSafepay into your X-Cart webshop.

This plugin has support for:

1. AfterPay
2. Alipay
3. American Express
4. Bancontact
5. Bank transfer
6. Belfius
7. Direct Debit

15. ING Home'Pay
16. KBC
